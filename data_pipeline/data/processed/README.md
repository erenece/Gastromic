# Processed Data

The `places.csv` and `reviews.csv` files in this folder are not included in the
GitHub repository due to their size.

You can access the full datasets on Kaggle:

- [Gastromic - Processed Data (Places)](https://www.kaggle.com/datasets/sahrasutylolu/gastromic-processed-data-places)
- [Gastromic - Processed Data (Reviews)](https://www.kaggle.com/datasets/sahrasutylolu/gastromic-processed-data-reviews)

## Contents

- **places.csv** — venue information (address, category, location, rating, review count, price level)
- **reviews.csv** — venue reviews (author, rating, date, review text)
